📝 Instructions

  1. Start a WHIP server (e.g., mediamtx on port 8889)
  2. Configure OBS: Settings → Stream → Service: "WHIP Service"
  3. Set WHIP URL in OBS: http://localhost:8889/mystream/whip
  4. Click "Start Streaming" in OBS
  5. Click "Connect" button above to view the stream
